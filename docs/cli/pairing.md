# `atomicbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
atomicbot pairing list whatsapp
atomicbot pairing approve whatsapp <code> --notify
```
