# `atomicbot tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
atomicbot tui
atomicbot tui --url ws://127.0.0.1:18789 --token <token>
atomicbot tui --session main --deliver
```
