# `atomicbot setup`

Initialize `~/.atomicbot/atomicbot.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
atomicbot setup
atomicbot setup --workspace ~/.atomicbot/workspace
```

To run the wizard via setup:

```bash
atomicbot setup --wizard
```
